# نمودارها — Figures

پنج نمودار مقاله، هر کدام در دو فرمت `.png` (برای پیش‌نمایش) و `.pdf` (برای چاپ با کیفیت بالا).

## فهرست

| فایل | محتوا | جای استفاده در مقاله |
|---|---|---|
| `fig1_generalization_matrix.png` | ماتریس گرمایی تعمیم‌پذیری بین‌مولدی | شکل ۲، بخش ۵-۳ |
| `fig2_robustness_paraphrasing.png` | مقایسه دقت قبل/پس از بازنویسی | شکل ۳، بخش ۵-۴ |
| `fig3_error_analysis.png` | سه پنل: منبع، حوزه، طول | شکل ۱، بخش ۵-۲ |
| `fig4_confusion_matrix.png` | ماتریس درهم‌ریختگی پیش/پس از حمله | شکل ۴، بخش ۵-۴ |
| `fig5_performance_summary.png` | خلاصه عملکرد در چهار شرایط | شکل ۵، بخش ۵-۵ |

## بازتولید

اگر می‌خواهید نمودارها را با عددهای به‌روز بازسازی کنید:

```bash
# روی Colab، نوت‌بوک ۶ را اجرا کنید — فایل‌های JSON در results/ کافی است
experiments/06_make_figures.ipynb
```

این نوت‌بوک نیاز به مدل، GPU یا اجرای دوباره آزمایش‌ها ندارد. تنها از پنج فایل JSON در `results/` می‌خواند.

## انتخاب طراحی

**چرا لیبل‌های انگلیسی روی نمودار فارسی‌محور؟** matplotlib بدون پکیج‌های اضافی (`arabic-reshaper`, `python-bidi`) فونت فارسی را جدا می‌نویسد و خط می‌اندازد. در عوض، **لیبل انگلیسی + توضیح فارسی در زیرنویس مقاله** رویه استاندارد ژورنال‌های فارسی مهندسی کامپیوتر است که در همه نشریات معتبر دیده می‌شود.

**چرا دو فرمت؟** `.png` در ۳۰۰dpi برای پیش‌نمایش و ضمیمه Word، و `.pdf` برداری برای کیفیت بالاتر در چاپ نهایی.
