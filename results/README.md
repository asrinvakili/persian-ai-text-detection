# نتایج آزمایش‌ها — Results

پنج فایل JSON که خروجی همه آزمایش‌های مقاله را به‌صورت ماشین‌خوان نگه می‌دارند.

| فایل | منبع | محتوا |
|---|---|---|
| `baseline_results.json` | نوت‌بوک ۱ | متریک‌های کلی baseline (دقت، F1، precision، recall) |
| `error_analysis.json` | نوت‌بوک ۲ | دقت به تفکیک منبع، حوزه موضوعی، طول متن |
| `train_on_one_matrix.json` | نوت‌بوک ۳ | ماتریس ۳×۳ آزمایش تک‌مولدی |
| `train_on_two_matrix.json` | نوت‌بوک ۴ | سه آزمایش leave-one-out دومولدی |
| `robustness_results.json` | نوت‌بوک ۵ | اثر حمله بازنویسی روی baseline |

## بارگذاری در پایتون

```python
import json

with open('results/baseline_results.json', encoding='utf-8') as f:
    baseline = json.load(f)

print(baseline['test_metrics']['eval_accuracy'])  # 0.9775
```

## یادداشت‌های راستی‌آزمایی

- اعداد فایل‌های ۱ و ۲ مستقیماً از خروجی Trainer و تحلیل پیش‌بینی‌ها استخراج شده‌اند.
- فایل ۵ شامل ماتریس درهم‌ریختگی نهایی (`confusion_matrix_after`) است که نشان می‌دهد چه تعدادی از ۶۰۰ متن AI بازنویسی‌شده به‌اشتباه انسانی طبقه‌بندی شدند (۳۹۰ از ۶۰۰).
- فایل‌های ۳ و ۴ ماتریس کامل دقت در هر شرایط آموزش × مولد آزمون را گزارش می‌کنند، که قلب یافته اصلی مقاله است.
